# 📱 Cellphone Manager

API REST desenvolvida em **Spring Boot** para gerenciamento de **celulares** e **marcas**.  
Permite realizar operações de CRUD e consultas filtradas por modelo ou marca e venda de telemóveis.

---

## ⚙️ Tecnologias Utilizadas
- **Java 17**
- **Spring Boot 2.7+**
- **Spring Data JPA**
- **Hibernate**
- **ModelMapper**
- **Validation API (Jakarta Validation)**
- **Banco de dados H2 / PostgreSql (dependendo do profile)**
- **Docker**

#### Ao dar run na API acesse a URL da documentação: http://localhost:8080/swagger-ui/index.html
